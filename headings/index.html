Typography

typography contains various heading like h0, h1, h2, h3, h4 & markdown.
Package Name:
fastn-community.github.io/typography

Overview

The typography package is already included inside doc-site. If you are using doc-site then don't add typography as fastn dependency.

How to use

Add below code to your FASTN.ftd
-- fastn.dependency: fastn-community.github.io/typography
Lang:
ftd
Import typography into your .ftd file
-- import: fastn-community.github.io/typography as tf
Lang:
ftd
Add typography components into your .ftd file
-- tf.h0: This is heading hero

Heading also has optional body property.

-- tf.h1: This is heading large

Heading also has optional body property.

-- tf.h2: This is heading medium

Heading also has optional body property.

-- tf.h3: This is heading small

Heading also has optional body property.

-- tf.h4: This is heading tiny

Heading also has optional body property.
Lang:
ftd
