Typography

markdown markdown is used to add body size content to your fastn pages.
Package Name:
fastn-community.github.io/typography

Overview

The typography package is already included inside doc-site. If you are using doc-site then don't add typography as fastn dependency.

How to use

Add below code to your FASTN.ftd and Import typography into your .ftd file
-- fastn.dependency: fastn-community.github.io/typography
Lang:
ftd
Import typography into your .ftd file
-- import: fastn-community.github.io/typography as tf
Lang:
ftd
Add markdown component into your .ftd file
-- tf.markdown:

This is markdown text. You can add [link](https://fastn.com) and
add **bold**, *italic* etc styles to markdown text.
Lang:
ftd

Example

Properties:
